Click on the button Calculate will create a request to get data from https://run.mocky.io/v3/d4e997fe-43ae-436d-b60c-31c305399520 using ajax request. You can use whatever library you like for fetching data (fetch, axios, JQuery...).

After data is received, you will use selected actions on it. All actions are happening one after another. So, you will first filter a primary collection (data from ajax request), then do an action on filtered collection, and then display a final collection. One action from each section has to be used.

    Sections:
  • Filter - returns all, even ([1,2,3,4,...] => [2,4,...]) or odd ([1,2,3,4,...] => [1,3,...]) numbers
  • Action - None - does not change a collection, Increase by 3 - inreases each numbers in a collection by 3 ([1,2,...] => [4,5,...]), Multiply by 2 - multiplies each number in a collection with 2 ([1,2,...] => [2,4,...])
  • Display - Displays a collection using console.log or alert
